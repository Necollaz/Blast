# Blast

Prototype of a Blast puzzle game made with TypeScript and Cocos Creator 2.4.x.

## Play

GitHub Pages: GitHub Pages: https://necollaz.github.io/Blast/

## Stack

- TypeScript
- Cocos Creator 2.4.x
- Web Mobile build

## Implemented

- N x M game board
- Random colored tiles
- Group detection by adjacent tiles
- Tile destruction
- Gravity and refill
- Score system
- Moves limit
- Win and lose states
- No available moves check
- Auto shuffle when no moves are available
- Bomb booster
- Teleport booster
- Super tiles
- Desktop and mobile browser support

## Run Project

1. Install Cocos Creator 2.4.x.
2. Clone the repository.
3. Open the project folder in Cocos Creator.
4. Open `assets/gameBlast/scenes/Game.fire`.
5. Run preview or build Web Mobile.

## Build

The published build is generated from Cocos Creator:

- Platform: Web Mobile
- Design resolution: 1080x1920
- Fit Width: enabled
- Fit Height: enabled
